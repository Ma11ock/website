Setup

My Setup

Priorities:

I am a Emacs user first and foremost. Because Emacs is essentially a Userland replacement all of the other programs I use have to conform to Emacs and not get in its way. This is why I no longer use tiling window managers: Emacs is already a window manager, there is no reason to use two programs with the same functionality.

Basic Setup

Text Editor

Emacs. I use Wordstar keybindings. In instances when I cannot run Emacs or it is too cumbersome for a task I use Joe's Own Editor.

Operating System

I run Debian Bullseye on my workstation these days. I ran Arch Linux and Manjaro but Manjaro became unstable after a year of use. My ThinkPad T420 is, however, still on Arch.

Desktop

XFCE is my desktop, with picom as its compositing window manager. KDE is probably objectively better but XFCE's killer feature is its customizability. It cannot be beat.

Terminal

Luke Smith's build of st. It is a preconfigured and patched version of the suckless terminal. I'm experimenting with vterm in Emacs but it is not quite ready to become a full-time terminal yet.

Email

msmtp, isync, and mu/mu4e so I can manage my mail in Emacs. I use gnome-keyring to manage my passwords.

Currently I am still stuck on GMAIL but I plan to move to my own mail server eventually...

Content

I consume content mostly via my RSS feed with elfeed. You can view my RSS feed here.

Documents

I use Latex with the LuaLatex engine for creating my own documents. I use Emacs's pdf-view-mode with auto-revert-mode enabled for a live environment. For pdf consumption I usually use Zathura or Gnome Document Viewer if I want something bloated.

Browser

Brave is the best in the business. Unfortunately, it seems to have problems on my workstation where it stutters while trying to open new tabs. Apparently this is because Brave tries to move massive amounts of data between main memory and VRAM. I retry Brave every six or so months to see if this is fixed and although it is better every time it still suffers from stuttering.

As a result, I am stuck on firefox.

Shell

zsh with oh my zsh as its package manager.

Emacs

Keybinds

I have Wordstar keybindings provided by my very own Joestar. It is still in pre-alpha so I do not recommend using it yet.

Theme

Modus Vivendi.

General

I've used Ivy since the beginning but I plan to get rid of it soon. I briefly used Helm but it's so bloated it slowed down my ThinkPad.

Themes

GTK and QT

For Icons and general GTK3 I use flat-remix-blue-dark. I use Qogir dark for my window theme.

Terminal

Just simple black-on-grey-text.

Font

Terminal font is Inconsolata but Emacs uses Hack. My UI font is usually Ubuntu Regular.
